<h2>一、项目简介</h2>
基于ssm的音乐播放平台管理系统：前端vue、element-ui，后端 spring、mybatis，集成歌单、歌手、我的音乐、我的收藏、音乐播放等功能于一体的系统。

<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">
<div class="markdown-heading" dir="auto">
<h3 dir="auto" tabindex="-1">管理员</h3>
<ul dir="auto">
 	<li>歌手管理：歌手信息的增删改查、歌手图片上传更新、批量删除、通过歌手信息管理歌曲</li>
 	<li>歌曲管理：歌曲信息的增删改查、歌单图片上传更新、歌单内容信息查看和修改、歌单评论列表查看</li>
 	<li>用户管理：用户信息的增删改查、用户收藏歌曲列表、用户收藏歌曲删除</li>
 	<li>系统首页：用户总数、歌曲总数、歌手数量、用户性别比例饼图、歌曲类型分布柱状图等</li>
</ul>
<h3 dir="auto" tabindex="-1"><a id="user-content-用户" class="anchor" tabindex="-1" href="https://github.com/YuLin-Coder/No56MusicPlayingPlatformManagementSystem?tab=readme-ov-file#%E7%94%A8%E6%88%B7" aria-hidden="true"></a>用户</h3>
<ul dir="auto">
 	<li>基本功能：登录、注册、退出</li>
 	<li>首页：轮播图展示、歌单列表推荐、歌手列表推荐、搜索音乐、主导航</li>
 	<li>歌单：分页展示全部歌单列表信息、支持分类浏览歌单、歌单详情、歌单评价、歌单发布评论</li>
 	<li>歌手：分页展示全部歌手信息、支持男歌手和女歌手分类浏览</li>
 	<li>我的音乐：浏览歌曲是可以收藏、收藏音乐列表</li>
 	<li>歌曲播放：选择歌曲、播放控制、开始、暂停、上一首、下一首</li>
</ul>
